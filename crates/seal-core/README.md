# seal-core

Core domain logic for [seal](https://github.com/bobisme/seal) — events, append-only event log storage, SQLite projection, and Git/jj SCM adapters.

This crate is the engine behind the `seal` CLI. Most users want [`seal-cli`](https://crates.io/crates/seal-cli) instead.

See the [main README](https://github.com/bobisme/seal#readme) for an overview of what seal does.
