# seal-tui

Terminal UI for [seal](https://github.com/bobisme/seal) — interactive review browser with syntax-highlighted diffs and inline commenting, built on [ftui](https://crates.io/crates/ftui-core).

This crate is consumed by the `seal` binary. Most users want [`seal-cli`](https://crates.io/crates/seal-cli) instead.

See the [main README](https://github.com/bobisme/seal#readme) for an overview of what seal does.
